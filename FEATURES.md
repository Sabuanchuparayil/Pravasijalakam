# Pravasi Jaalakam - Feature Roadmap

## V1 (MVP) - Core Features

### 1. Authentication & User Management
- [x] User registration and login (Clerk.dev)
- [x] User profiles
- [x] Role-based access control (Guest, Member, Author, Admin)
- [ ] Email verification
- [ ] Password reset

### 2. Literature Publishing & Reading
- [x] Create, edit, delete literature (stories, poems, articles)
- [x] Draft and publish workflow
- [x] Literature listing with filters (type, language, author, tags)
- [x] Search functionality (basic)
- [ ] Reading experience (typography, font controls)
- [ ] Bookmarking
- [ ] Reading history
- [ ] Featured content

### 3. Author Profiles
- [x] Author profile creation
- [x] Author verification system
- [x] Author pages with works
- [ ] Author statistics (views, likes, followers)
- [ ] Author badges/recognition

### 4. Comments & Engagement
- [x] Comment on literature
- [x] Threaded comments (replies)
- [x] Like literature
- [ ] Edit/delete comments
- [ ] Comment moderation

### 5. Community Features
- [x] Classifieds (create, list, filter)
- [x] Events (create, list, RSVP)
- [ ] Event calendar view
- [ ] Classified categories
- [ ] Location-based filtering

### 6. Moderation
- [x] Report content
- [x] Report management (admin)
- [ ] Content moderation dashboard
- [ ] Automated moderation (basic)
- [ ] User blocking

### 7. Admin Dashboard
- [x] Report management
- [ ] User management
- [ ] Content moderation
- [ ] Author verification
- [ ] Analytics dashboard

### 8. Internationalization
- [x] Malayalam as primary language
- [x] English as secondary language
- [ ] Language switcher
- [ ] RTL support (if needed)
